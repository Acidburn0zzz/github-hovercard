GitHub Hovercard

What's GitHub Hovercard?

GitHub Hovercard provides neat user/repo/issue hovercards for GitHub. It brings information you may be interested with all into your current page.

Features

  • Extracts users, repos and issues almost everywhere around GitHub
  • Friendly emoji display in repo descriptions and issue contents
  • Markdown conversion for issue contents
  • Supports binding access token to increase API request limit
  • Supports GitHub's task lists in Markdown
  • Supports GitHub Enterprise
